# What Is CryptoKids NFT?

CryptoKids is an NFT avatar project — a collection of playful, generated profile-picture characters. It was the very first thing I ever shipped to the web, built long before I knew how to write code.

## Why I Built It

At the time, NFT avatar collections were everywhere, and I wanted to understand the space by actually launching something rather than just reading about it. The goal was less about the art and more about learning the full journey: putting up a landing page, telling a story around a collection, and getting it in front of people.

## How It Was Made

Since I couldn't code yet, I built the entire site with [Wix](https://www.wix.com), a drag-and-drop website builder. It let me design and publish a presentable page without touching HTML, CSS, or JavaScript.

## What It Taught Me

Looking back, CryptoKids is what nudged me toward learning to code properly. Hitting the limits of a no-code builder — the slow load times, the lack of control, the inability to add the features I imagined — is exactly what pushed me to pick up real web development. Every project after this one was built from scratch.
